# Sfinx

Sfinx is a virtual interview platform designed to assess the soft skills of candidates. By comparing a candidate's performance against a reference worker, Sfinx serves as an innovative HR recruitment tool to filter and streamline the initial interview process, saving valuable time for recruiters and hiring managers. It leverages ElevenLabs for its conversational AI capabilities.

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](httpshttps://vercel.com/font), a new font family for Vercel.
